Repositorio remoto, el que esta en la nube
Repositorio local, el que tenemos en el pc
git status, para ver el estado de los archivos
git commit, para confirmar los cambios 
git pull, descargar del rep-remoto los archivos/cambios
git push, subir archivos/cambios al rep-remoto
origin master/main = la rama donde subimos el proyecto
git config user.name, Para el nombre del usuario del github
git config user.email, Para el correo del github
git init, inicializar el repositorio local
git add, 

Los archivos sin rastrear son todos los demás - cualquier otro archivo en tu directorio de 
trabajo que no estaba en tu última instantánea y que no está en el área de preparación (staging area)

Archivos rastreados (tracked files) son todos aquellos archivos que estaban en la última 
instantánea del proyecto
